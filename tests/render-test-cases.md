# Render Test Cases

## Case 1: Basic Texture

Expected:
- Object displays diffuse texture without lighting.

## Case 2: Lambert Lighting

Expected:
- Brightness changes according to light direction.

## Case 3: Blinn-Phong

Expected:
- Specular highlight appears when view direction and light direction align.

## Case 4: Toon Shading

Expected:
- Light intensity appears in discrete bands.

## Case 5: Normal Mapping

Expected:
- Surface detail changes according to normal map.
