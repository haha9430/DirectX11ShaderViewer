# Portfolio Summary

## DirectX 11 기반 3D 셰이더 렌더링 뷰어

DirectX 11과 HLSL을 사용하여 실시간 3D 렌더링 데모를 구현했습니다.  
Vertex Shader와 Pixel Shader를 직접 작성하고, Constant Buffer를 통해 월드/뷰/투영 행렬과 조명 데이터를 GPU에 전달했습니다.

주요 기능으로는 텍스처 매핑, Lambert 조명, Blinn-Phong 조명, Rim Lighting, Toon Shading, Normal Mapping, ImGui 기반 실시간 파라미터 조절 UI가 있습니다.

이를 통해 DirectX 11 렌더링 파이프라인, 셰이더 프로그래밍, 조명 모델, 디버그 UI 구성 경험을 쌓았습니다.
